- scikit-learn

## Project Structure
- python_basics  
  Basic numerical computing with NumPy and ColabTurtle.

- data_analysis

- machine_learning  
  Training ML models using scikit-learn.

- nlp  
  Natural language processing using spaCy and NLTK.

- web_scraping  
  Web scraping using BeautifulSoup.
